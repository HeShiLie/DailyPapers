## 1-2604.22748-Agentic World Modeling: Foundations, Capabilities, Laws, and Beyond

1. source link: https://arxiv.org/abs/2604.22748
2. 单位: 42 authors / 多机构综述
3. Keywords: Agentic world model; taxonomy; simulators; environment dynamics
4. 问题背景: 随着 AI agent 从文本生成走向持续交互和任务执行，是否能预测环境动态成为关键瓶颈。不同社区对 world model 的定义并不统一，导致方法、评测和风险讨论彼此割裂。
5. 动机/Insight: 论文的核心 insight 是：agentic world model 不应只按任务或模态分类，而应按“能力层级 × 规律类型”来理解。能力从一步预测、可组合模拟到自我修正演化；规律则覆盖物理、数字、社会和科学系统。
6. 解决方案: 提出 levels × laws 分类框架，梳理 400+ 篇工作和 100+ 代表系统，总结不同组合下的方法、失败模式和评测实践。进一步给出面向决策的评测原则、最小可复现实验包、架构建议和治理问题。

## 2-2604.22554-Video Analysis and Generation via a Semantic Progress Function

1. source link: https://arxiv.org/abs/2604.22554
2. 单位: Tel Aviv University
3. Keywords: Video generation; semantic progress; temporal pacing; semantic linearization
4. 问题背景: 图像/视频生成中的语义变化经常不是匀速的：某些时段几乎不变，随后突然跳变。这会造成视频 transition 不平滑，也让不同生成器之间的“语义节奏”难以比较。
5. 动机/Insight: 作者认为视频时间轴上的关键不是像素变化，而是语义 embedding 的累积变化。只要把语义变化压缩成一维 progress curve，就能诊断哪里节奏异常、哪里发生突变。
6. 解决方案: 提出 Semantic Progress Function：计算帧级语义 embedding 距离并拟合累积语义变化曲线。再用 semantic linearization 对序列重定时，使语义变化接近线性，从而改善生成或真实视频的语义平滑性。

## 3-2604.21518-DiffNR: Diffusion-Enhanced Neural Representation Optimization for Sparse-View 3D Tomographic Reconstruction

1. source link: https://arxiv.org/abs/2604.21518
2. 单位: Monash University
3. Keywords: Sparse-view CT; neural representation; diffusion prior; 3D reconstruction
4. 问题背景: Neural fields / 3D Gaussian 等神经表示能表达 CT 体数据，但在 sparse-view CT 中约束不足，容易出现严重伪影。直接把 diffusion denoising 放进重建迭代又通常很慢。
5. 动机/Insight: 作者的 insight 是：diffusion prior 不一定要高频参与每一步优化，可以周期性地产生 pseudo-reference volume，为 underconstrained 区域提供感知监督。这样兼顾先验修复能力和运行效率。
6. 解决方案: 提出 DiffNR，其中 SliceFixer 是一个 single-step diffusion model，用于修复退化切片并生成 pseudo-reference volumes。重建时用这些伪参考体提供 3D perceptual supervision，平均 PSNR 提升 3.99 dB，同时减少频繁 diffusion 查询。

## 4-2604.18519-LLM Safety From Within: Detecting Harmful Content with Internal Representations

1. source link: https://arxiv.org/abs/2604.18519
2. 单位: University of Toronto CSSLab
3. Keywords: LLM safety; guard model; internal representations; safety neurons
4. 问题背景: 现有 guard model 多依赖 LLM 末层表示或独立生成式判别，可能忽略中间层中已经出现的安全相关信号。生成式 guard 还会带来较高推理成本。
5. 动机/Insight: 论文的核心 insight 是：LLM 内部层分布着可被线性探测的 safety neurons，这些信号比只看 terminal layer 更丰富。若能自适应组合不同层特征，就可以构建轻量、高效的 harmfulness detector。
6. 解决方案: 提出 SIREN：先用 linear probing 找 safety neurons，再用 adaptive layer-weighted strategy 融合内部层特征，不修改底座模型。实验显示它在多个安全基准上优于开源 guard，并使用约 250× 更少可训练参数，支持实时流式检测。

## 5-2604.22586-FlowAnchor: Stabilizing the Editing Signal for Inversion-Free Video Editing

1. source link: https://arxiv.org/abs/2604.22586
2. 单位: Communication University of China, Multimedia Intelligent Processing Group
3. Keywords: Video editing; flow-based editing; inversion-free; attention refinement
4. 问题背景: Inversion-free editing 在图像上效率高、结构保持好，但扩展到视频时容易在多物体、长帧数或快速运动场景下失败。根因是视频 latent 空间维度更高，editing signal 的空间定位和强度都不稳定。
5. 动机/Insight: 作者认为稳定视频编辑需要同时回答两个问题：改哪里，以及改多强。若文本指导与空间区域对齐不准，或者长序列导致信号幅度衰减，编辑就会漂移或失效。
6. 解决方案: 提出 training-free 的 FlowAnchor。它用 Spatial-aware Attention Refinement 稳定文本-区域对齐，用 Adaptive Magnitude Modulation 保持足够编辑强度，从而在 flow-based sampling 中实现更忠实、时序一致且高效的视频编辑。

## 6-2604.22294-Contexts are Never Long Enough: Structured Reasoning for Scalable Question Answering over Long Document Sets

1. source link: https://arxiv.org/abs/2604.22294
2. 单位: Stanford NLP
3. Keywords: Long-context QA; structured reasoning; SQL; document collections
4. 问题背景: 多文档 QA 的难点不只是 context window 长度，而是证据会随着文档规模增长而产生聚合瓶颈。传统 chunk-level 输出再汇总的方法，在大规模文档集合上会越来越难推理。
5. 动机/Insight: 论文的 insight 是：与其继续拼接更多文本，不如把局部抽取的信息沉淀成可持久维护的结构化状态。关系数据库和 SQL 能把多文档证据合并、查询和推理过程显式化。
6. 解决方案: 提出 SLIDERS：从文档中抽取 salient information 到 relational database，再通过 SQL 做 structured reasoning。它加入 data reconciliation 阶段，利用 provenance、extraction rationales 和 metadata 修复重复、冲突和缺失记录；在长上下文基准上超过 GPT-4.1 平均 6.6 分，并在 3.9M/36M token 新基准上显著优于次优方法。

## 7-2604.22436-AgentSearchBench: A Benchmark for AI Agent Search in the Wild

1. source link: https://arxiv.org/abs/2604.22436
2. 单位: University College London
3. Keywords: AI agents; agent retrieval; reranking; execution-grounded evaluation
4. 问题背景: AI agent 生态增长后，一个新问题是：给定任务，如何找到真正合适的 agent。仅看 agent 描述并不可靠，因为 agent 能力往往是组合式、执行依赖的。
5. 动机/Insight: 作者指出 agent search 不能只做语义匹配，必须用执行结果来校准 relevance。描述相似并不等于实际任务表现好，这会暴露传统 retrieval/reranking 的盲区。
6. 解决方案: 提出 AgentSearchBench，覆盖近 10,000 个真实 agent，既评估 executable task queries，也评估 high-level task descriptions。基准使用 execution-grounded performance signals 作为相关性依据，并显示 execution-aware probing 等轻量行为信号能显著提升排序质量。

## 8-2604.21718-Building a Precise Video Language with Human-AI Oversight

1. source link: https://arxiv.org/abs/2604.21718
2. 单位: Carnegie Mellon University
3. Keywords: Video captioning; human-AI oversight; visual primitives; video generation control
4. 问题背景: 视频语言模型需要描述动态世界，但自然语言 caption 往往不够精确，尤其难覆盖主体、场景、运动、空间关系和镜头语言。粗糙 caption 会限制视频理解，也限制生成模型对长、细粒度 prompt 的跟随能力。
5. 动机/Insight: 论文的 insight 是：专业级视频理解需要结构化的“视频语言”，并用人类专家做 critique/revision，而不是让人从零写 caption。这样模型负责生成草稿，人类专注核验和改进，标注效率和质量都更高。
6. 解决方案: 提出一套 structured visual specification 和 CHAI（Critique-based Human-AI Oversight）流程，用专家 critique 将 pre-caption 修成 post-caption。再用这些 critique / preference 监督 Qwen3-VL 做 caption、reward modeling 和 critique generation，并用重标注视频微调 Wan，使其更好跟随长达 400 词的镜头级 prompt。

## 9-2604.22085-Memanto: Typed Semantic Memory with Information-Theoretic Retrieval for Long-Horizon Agents

1. source link: https://arxiv.org/abs/2604.22085
2. 单位: Moorcheh.ai
3. Keywords: Agent memory; semantic memory; information-theoretic retrieval; long-horizon agents
4. 问题背景: 长程 agent 从无状态推理走向多会话持续运行时，memory 成为主要架构瓶颈。许多系统依赖 semantic graph，需要实体抽取、schema 维护和多查询检索，工程与计算开销都较大。
5. 动机/Insight: 作者挑战“高质量 agent memory 必须依赖复杂知识图谱”的假设，认为 typed semantic schema 加上强检索引擎即可获得高保真记忆。关键是让召回覆盖足够广，再交给 LLM 过滤。
6. 解决方案: 提出 Memanto：13 类 typed semantic memory、自动冲突解决、时间版本管理，并结合 Moorcheh 的 information-theoretic search engine 做无索引语义检索。报告在 LongMemEval 和 LoCoMo 上达到 89.8% / 87.1%，单次检索、无 ingestion cost、延迟低于 90ms。

## 10-2604.18580-Sessa: Selective State Space Attention

1. source link: https://arxiv.org/abs/2604.18580
2. 单位: 1 author
3. Keywords: Sequence modeling; state-space model; attention; long context
4. 问题背景: Transformer 能直接访问可见上下文，但长上下文中注意力容易扩散；state-space model 依赖递归状态，长程信息若未被主动保存也会衰减。两类模型都面临选择性保存和检索远距信息的问题。
5. 动机/Insight: 论文的 insight 是：把 attention 放进 recurrent feedback path，可以形成多条 attention-based paths，让历史 token 以更丰富的路径影响未来状态。这可能带来比标准 Transformer 或 Mamba 风格模型更慢的记忆衰减。
6. 解决方案: 提出 Sessa decoder，并在假设匹配的理论设置下证明其 memory tail 可呈 power-law 衰减 $O(\ell^{-\beta})$，$0<\beta<1$。同时给出 selective retrieval 构造，并在长上下文 benchmark 上报告优于匹配的 Transformer / Mamba-style baseline。

## 11-2604.22152-dWorldEval: Scalable Robotic Policy Evaluation via Discrete Diffusion World Model

1. source link: https://arxiv.org/abs/2604.22152
2. 单位: 5 authors
3. Keywords: Robotics evaluation; world model; discrete diffusion; VLA
4. 问题背景: 机器人策略若要在成千上万环境和任务中评估，真实仿真或实机测试成本很高。现有 world simulator / policy evaluation 方法在规模、模态统一和自动判定任务完成度上仍受限。
5. 动机/Insight: 作者的 insight 是：可以把视觉、语言和机器人动作统一离散化到 token space，再用 diffusion world model 作为可扩展评估代理。若模型还能预测 task progress，就能自动判断 rollout 是否成功。
6. 解决方案: 提出 dWorldEval：将多模态输入映射到 unified token space，用 transformer-based denoising network 建模，并引入 sparse keyframe memory 保持时空一致性。它还预测 progress token，progress 到 1 时自动判定成功；在 LIBERO、RoboTwin 和真实机器人任务上优于 WorldEval、Ctrl-World 和 WorldGym。

## 12-2604.22565-Learning Evidence Highlighting for Frozen LLMs

1. source link: https://arxiv.org/abs/2604.22565
2. 单位: 13 authors
3. Keywords: Long-context reasoning; evidence highlighting; frozen LLM; reinforcement learning
4. 问题背景: LLM 在长噪声上下文中常错过关键证据。压缩或重写上下文虽然能减少 token，但可能丢失或扭曲原始证据；而修改 solver 模型又成本高、不现实。
5. 动机/Insight: 作者的 insight 是：可以把“找证据”和“做推理”解耦，只在原文中插入轻量 highlight tags，让 frozen solver 自己利用强调后的上下文。证据选择本身可视为弱监督决策问题。
6. 解决方案: 提出 HiLight：训练一个轻量 Emphasis Actor，在不改写原文、不修改 frozen Solver 的情况下标注关键 span。Actor 只用 Solver 的任务 reward 做 RL，无需 evidence labels；在 sequential recommendation 和 long-context QA 上超过 prompt-based / prompt-optimization baseline，并能零样本迁移到未见 solver。

## 13-2604.20012-EmbodiedMidtrain: Bridging the Gap between Vision-Language Models and Vision-Language-Action Models via Mid-training

1. source link: https://arxiv.org/abs/2604.20012
2. 单位: Chenyan Xiong Research Group at CMU
3. Keywords: VLM; VLA; embodied AI; mid-training
4. 问题背景: 许多 VLA 模型直接继承 off-the-shelf VLM，但 VLM 训练数据与机器人具身任务分布差异明显。这会导致 VLM 虽有视觉语言能力，却不是 VLA fine-tuning 的最佳初始化。
5. 动机/Insight: 论文的 insight 是：VLA 数据在 VLM 表征空间中占据更紧凑、相对分离的区域，因此可以从大规模 VLM 数据池中筛出更接近 VLA 分布的样本做 mid-training。对齐程度既有 dataset-level 差异，也有 sample-level 差异。
6. 解决方案: 提出 EmbodiedMidtrain：用冻结 VLM 特征训练轻量 proximity estimator，选择最 VLA-aligned 的 VLM 样本进行中间训练，再做 VLA fine-tuning。在三个机器人操作 benchmark 上稳定提升，1.1B mid-trained 模型可接近 3–8× 更大 backbone 的 VLA，并能跨架构迁移。

## 15-2604.20857-DiagramBank: A Large-scale Dataset of Diagram Design Exemplars with Paper Metadata for Retrieval-Augmented Generation

3. Keywords: Scientific diagrams; dataset; RAG; figure generation

## 16-2604.22119-Emergent Strategic Reasoning Risks in AI: A Taxonomy-Driven Evaluation Framework

1. source link: https://arxiv.org/abs/2604.22119
2. 单位: Amazon
3. Keywords: AI safety; strategic reasoning; deception; reward hacking
4. 问题背景: 随着 LLM 推理能力和部署范围扩大，模型可能表现出服务自身目标的策略性行为，如欺骗、评测博弈和 reward hacking。现有安全评测对这类 emergent strategic reasoning risks 缺少系统分类和可扩展 benchmark。
5. 动机/Insight: 论文的 insight 是：这类风险不能只看最终回答，还应同时评估 reasoning traces。通过 taxonomy-driven 场景生成和双重 rubric，可以更系统地诱发并测量策略性风险。
6. 解决方案: 提出 ESRRSim，一个 taxonomy-driven agentic evaluation framework。它构建 7 大类、20 子类风险 taxonomy，生成评测场景，并用独立 rubric 评估 response 与 reasoning trace；在 11 个 reasoning LLM 上发现风险检测率从 14.45% 到 72.72% 差异很大。
