## 1-2604.04746-Think in Strokes, Not Pixels: Process-Driven Image Generation via Interleaved Reasoning

1. source link: arXiv: 2604.04746
    
2. 单位：AI at Meta
    
3. Keywords：process-driven generation, interleaved reasoning, step-wise supervision, image synthesis
    
4. 问题背景：传统文生图通常是一把生成，过程隐式且难解释，也难以在中间状态进行监督。作者关注的是，模型能否像人画画一样，先规划、再起稿、再反思、再细化。
    
5. 动机/Insight：核心想法是把图像生成拆成“文本思考 + 视觉状态演进”交替进行的多步过程。这样中间图像不再只是副产物，而是后续推理的显式条件。
    
6. 解决方案：提出四阶段循环：textual planning、visual drafting、textual reflection、visual refinement，并对中间文本与中间图像都施加 step-wise supervision，约束语义一致性与纠错能力。([Hugging Face](https://huggingface.co/papers/2604.04746 "Paper page - Think in Strokes, Not Pixels: Process-Driven Image Generation via Interleaved Reasoning"))
    

## 2-2604.06268-RAGEN-2: Reasoning Collapse in Agentic RL

1. source link: arXiv: 2604.06268
    
2. 单位：MLL Lab
    
3. Keywords：agentic RL, reasoning collapse, mutual information, SNR-aware filtering
    
4. 问题背景：多轮 agent RL 训练里，大家常用 entropy 监控 reasoning stability，但 entropy 只能看“同一输入下的多样性”，看不到 reasoning 是否真的依赖输入。
    
5. 动机/Insight：作者指出一种隐藏失效模式——template collapse：看似多样，实则对不同输入套同一模板。比起 entropy，cross-input distinguishability 才更接近真正的 reasoning quality。
    
6. 解决方案：提出基于 mutual information 的在线代理指标来诊断 reasoning collapse，并进一步从 SNR 角度解释该现象，随后用 SNR-Aware Filtering 选择高信号 prompt，提升 reasoning quality 与任务性能。([Hugging Face](https://huggingface.co/papers/2604.06268 "Paper page - RAGEN-2: Reasoning Collapse in Agentic RL"))
    

## 3-2604.07023-MARS: Enabling Autoregressive Models Multi-Token Generation

3. Keywords：autoregressive LM, multi-token generation, fine-tuning, KV cache
    
4. 问题背景：AR 语言模型一次只吐一个 token，即使后续 token 已经高度可预测，也仍然要逐步解码，吞吐受限。
