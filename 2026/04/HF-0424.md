## 1-`2604.17295`-`LLaTiSA: Towards Difficulty-Stratified Time Series Reasoning from Visual Perception to Semantics`

1. source link: https://arxiv.org/abs/2604.17295
2. 单位：AMAP-ML
3. Keywords：time series reasoning, VLM, hierarchical benchmark, curriculum fine-tuning
4. 问题背景：LLM/VLM 对时间序列的理解长期被碎片化任务和含糊 benchmark 限制，尤其难以同时处理图形模式、数值表格和语义推理。现有模型容易在视觉趋势识别与精确数值定位之间失衡，导致时间序列问答不稳定。
5. 动机/Insight：作者把 Time Series Reasoning 明确定义为四级认知复杂度，希望用难度分层来区分“看懂趋势”和“做复杂语义推理”。核心 insight 是：仅给图像不够，必须把可视化 pattern 与精确 index-value table 结合，降低数值幻觉。
6. 解决方案：提出 HiTSR，包含 83k 分层时间序列样本和 verified CoT；再基于此训练 LLaTiSA。模型通过多阶段 curriculum fine-tuning 融合图形视觉模式和校准后的数值表格，在 TSR 和 OOD 场景上提升表现。

## 2-`2604.21686`-`WorldMark: A Unified Benchmark Suite for Interactive Video World Models`

1. source link: https://arxiv.org/abs/2604.21686
2. 单位：上海AI Lab，东京大学
3. Keywords：interactive video world model, benchmark, control alignment, world consistency
4. 问题背景：交互式视频世界模型发展很快，但 Genie、YUME、HY-World、Matrix-Game 等系统通常各用各的场景、动作和评价方式。这样导致模型间很难公平比较，尤其是控制一致性和世界一致性的评估不可复现。
5. 动机/Insight：作者认为 world model benchmark 的关键不是单一指标，而是“相同场景 + 相同动作序列 + 统一控制接口”。只有把输入条件标准化，视觉质量、轨迹误差和 VLM judge 才有可比性。
6. 解决方案：提出 WorldMark：用统一 WASD-style action vocabulary 映射到不同模型原生控制；构造 500 个覆盖第一/第三人称、写实/风格化、Easy-Hard 难度的测试用例；并提供视觉质量、控制对齐、世界一致性三类模块化评估工具。

## 3-`2604.19734`-`UniT: Toward a Unified Physical Language for Human-to-Humanoid Policy Learning and World Modeling`

1. source link: https://arxiv.org/abs/2604.19734
2. 单位：XPENG Robotics
3. Keywords：humanoid policy learning, latent action tokenizer, cross-embodiment transfer, world modeling
4. 问题背景：类人机器人 foundation model 的瓶颈是机器人数据稀缺，而人类 egocentric 数据虽然多，却和 humanoid 存在运动学差异。直接从人类动作迁移到机器人策略或世界模型会遇到 embodiment gap。
5. 动机/Insight：作者的核心假设是：不同 embodiment 的运动学不同，但其视觉后果中存在共享的物理意图。若能把人类动作和 humanoid 动作映射到同一个离散 latent physical language，就能借人类数据提升机器人策略和视频世界模型。
6. 解决方案：提出 UniT，通过 visual anchoring 和 tri-branch cross-reconstruction：action 预测视觉以对齐物理结果，vision 重建 action 以过滤无关视觉因素，并由 fusion branch 形成 embodiment-agnostic discrete tokens。该 token 同时用于 VLA policy learning 和 world modeling，实现人到 humanoid 的策略/动作迁移。

## 5-`2604.20987`-`Co-Evolving LLM Decision and Skill Bank Agents for Long-Horizon Tasks`

3. Keywords：LLM agents, skill bank, long-horizon tasks, co-evolution
4. 问题背景：长程交互环境要求 agent 能跨多步组合技能、处理 delayed reward 和 partial observability。LLM agent 虽然能推理，但通常缺少跨 episode 发现、保存和复用结构化技能的机制。

## 9-`2604.20244`-`Hybrid Policy Distillation for LLMs`

1. source link: https://arxiv.org/abs/2604.20244
2. 单位：SJTU / Shanghai Innovation Institute / Tencent 等
3. Keywords：knowledge distillation, forward KL, reverse KL, on-policy sampling
4. 问题背景：LLM 蒸馏效果受 KL 方向、优化策略和数据来源共同影响。forward KL 强调覆盖 teacher 分布，reverse KL 更 mode-seeking；单独使用往往在稳定性、效率或最终性能上有取舍。
5. 动机/Insight：作者把多种 KD 方法统一成 token-level reweighted log-likelihood 视角，并认为应结合 forward/reverse KL 的互补性。再通过轻量近似 on-policy sampling 弥补纯 offline 数据带来的分布偏移。
6. 解决方案：提出 Hybrid Policy Distillation：混合 forward KL 与 reverse KL，并结合 off-policy data 与 lightweight approximate on-policy sampling。长生成数学推理、短生成对话和代码任务上提升优化稳定性、计算效率和最终效果。

## 11-`2604.10268`-`EditCrafter: Tuning-free High-Resolution Image Editing via Pretrained Diffusion Model`

3. Keywords：image editing, high resolution, tiled inversion, diffusion guidance

## 19-`2604.20156`-`Temporally Extended Mixture-of-Experts Models`

1. source link: https://arxiv.org/abs/2604.20156
2. 单位：Princeton University
3. Keywords：Mixture-of-Experts, options framework, expert switching, serving efficiency
4. 问题背景：MoE 模型通过稀疏专家扩展容量，但 token-level 频繁切换专家会造成专家加载/卸载开销。模型大到超出单机显存后，offloading 和 prefetching 会因切换过快而失效。
5. 动机/Insight：作者把 RL 中 options framework 用到 MoE：专家选择不必每个 token 都重新决策，而可以 temporally extended。通过 deliberation cost，可在能力和切换频率之间权衡。
6. 解决方案：在每层加入 controller，学习何时切换 expert set 以及加载哪些专家，并用 option-critic + deliberation cost 训练。基于 gpt-oss-20b、LoRA 和 self-distillation reward，将 switch rate 从 50%+ 降到 5% 以下，同时保留最多 90% base-model accuracy。

3. Keywords：EEG foundation model, test-time adaptation, distribution shift, benchmark
4. 问题背景：EEG foundation model 在大规模神经数据上有潜力，但临床部署会遇到设备、人群、场景等分布漂移。TTA 可在无源数据和无标注目标数据下推理时适配，适合隐私受限医疗场景，但在 EEG 中研究不足。
5. 动机/Insight：作者系统性评估 TTA 在 EEG foundation model 上是否真的有效。结果显示通用 TTA 方法并不稳定，尤其 gradient-based adaptation 容易退化。
6. 解决方案：提出 NeuroAdapt-Bench，覆盖多个预训练 EEG foundation models、多下游任务和异质数据集，包括 in-distribution、OOD 和 Ear-EEG 等极端 modality shift。实验发现 optimization-free 方法更稳定，提示 EEG 需要 domain-specific adaptation。

## 21-`2604.08645`-`3D-VCD: Hallucination Mitigation in 3D-LLM Embodied Agents through Visual Contrastive Decoding`

1. source link: https://arxiv.org/abs/2604.08645
2. 单位：Perception and LANguage Lab @ UIUC
3. Keywords：3D-LLM, embodied agents, hallucination mitigation, contrastive decoding
4. 问题背景：多模态大模型正被用于 3D embodied agent 推理，但容易产生物体存在、空间布局和几何 grounding 相关幻觉。现有 2D VLM hallucination mitigation 方法难以直接迁移到 3D 场景。
5. 动机/Insight：作者认为 3D 幻觉应在结构化 3D scene graph 上做 contrastive decoding，而不是仅扰动像素。若模型在原始和失真 3D 上都倾向输出某些 token，这些 token 更可能来自语言先验而非场景证据。
6. 解决方案：提出 3D-VCD：构造带语义/几何扰动的 distorted 3D scene graph，如类别替换、坐标或 extent corruption；然后对比原始与扰动上下文的预测，抑制不受场景证据影响的 token。方法无需训练，在 3D-POPE 和 HEAL 上降低幻觉并提升 grounded reasoning。
