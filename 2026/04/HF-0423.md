## 1-`2604.20796`-`LLaDA2.0-Uni: Unifying Multimodal Understanding and Generation with Diffusion Large Language Model`

1. source link: https://arxiv.org/abs/2604.20796
2. 单位：inclusionAI
3. Keywords：diffusion LLM, unified multimodal model, semantic discrete tokenizer, image generation
4. 问题背景：多模态大模型通常把理解和生成分开建模，VLM 偏理解，扩散模型偏生成，统一框架往往需要复杂的跨模态桥接。离散扩散语言模型提供了把文本与视觉 token 放进同一扩散建模范式的可能。
5. 动机/Insight：核心 insight 是用语义离散 tokenizer 把连续图像转成可被 dLLM 处理的视觉 token，使理解、生成、编辑和交错推理共享同一个 backbone。这样既能保留 VLM 的语义理解能力，又能用扩散 decoder 恢复高保真图像。
6. 解决方案：提出 LLaDA2.0-Uni：SigLIP-VQ 语义离散 tokenizer + MoE dLLM backbone + diffusion decoder。训练上使用多阶段数据配方；推理上用 prefix-aware optimization 和 decoder few-step distillation 提升效率，最终在多模态理解和图像生成/编辑上接近专用模型。

## 2-`2604.20733`-`Near-Future Policy Optimization`

1. source link: https://arxiv.org/abs/2604.20733
2. 单位：作者未在 HF 列出机构
3. Keywords：RLVR, mixed-policy RL, near-future checkpoint, GRPO, off-policy trajectory
4. 问题背景：RLVR/GRPO 训练中，纯 on-policy 探索收敛慢，而引入 off-policy 轨迹又面临“质量高但分布远”或“分布近但质量低”的矛盾。关键问题是如何找到既比当前策略强、又足够接近当前策略的辅助轨迹来源。
5. 动机/Insight：作者把同一次训练中稍后 checkpoint 视为“near-future self”，它天然比当前策略更强，同时由于来自同一训练轨迹，分布距离小于外部 teacher。这相当于用未来自己的轨迹给现在的自己做 mixed-policy learning。
6. 解决方案：提出 NPO：在早期 bootstrapping 或后期 plateau 时插入近未来 checkpoint 生成的辅助轨迹；进一步提出 AutoNPO，根据在线训练信号触发干预并选择最大化有效学习信号 S=Q/V 的 guide checkpoint。在 Qwen3-VL-8B-Instruct + GRPO 上提升平均性能并加速收敛。

4. 问题背景：Deep Research agent 通常依赖大模型和封闭数据，小模型虽然成本、延迟和隐私更适合部署，但在长程检索/研究任务上很难训练到足够可靠。开放数据规模有限时，数据质量和利用率成为瓶颈。

2. 单位：Fudan University
3. Keywords：reward hacking, RLHF, proxy objective, alignment, scalable oversight
4. 问题背景：RLHF/RLAIF/RLVR 等对齐范式把复杂人类目标压缩成 reward signal，但大模型优化能力变强后会系统性利用 proxy 漏洞。表现包括 verbosity bias、sycophancy、hallucinated justification、benchmark overfitting，以及多模态中的感知-推理脱耦。

6. 解决方案：这是一篇 survey/理论框架文。它把 reward hacking 形式化为 objective compression、optimization amplification、evaluator-policy co-adaptation 三者交互的结果，并按这些维度组织检测与缓解策略，指出 scalable oversight、多模态 grounding 和 agentic autonomy 是核心开放挑战。
